# Who Are You Between the Pages?
<img width="1003" height="638" alt="image" src="https://github.com/user-attachments/assets/814e07ab-11be-4e2e-a453-c298c4a30722" />

A web-based personality quiz that matches you to a literary character based on your values, personality, and behavior. Answer 10 questions and discover which character from My Name Is Asher Lev, Between Two Moons, or When Mystical Creatures Attack! you most resemble.
Features

10 personality-based questions with weighted scoring
Automatic tiebreaker system for close results
Character result with description and book cover
Bar chart showing your full personality breakdown across all characters

Built With:
-----------
HTML, CSS, JavaScript
Chart.js for personality visualization

Demo:
-----------
Answer questions → Get matched to a character → See your personality breakdown chart

About:
-----------
Built as a creative project connecting personality psychology with literary analysis. Each character was mapped to distinct personality traits, and the scoring system weighs answers to determine your closest match.
